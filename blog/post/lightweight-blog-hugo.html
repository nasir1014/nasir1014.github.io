輕量級落格平台建置篇：HUGO

Published on Sun, Oct 13, 2024 by Nasir

在這篇文章中，我們將探討如何使用 Hugo 來建置一個輕量級的部落格平台。Hugo 是一個快速的靜態網站生成器，非常適合用來建置個人網站或部落格。

Hugo 是什麼？

Hugo 就是一個 Markdown 語法建立部落格的工具，常見的部落格建置工具可能都是那種 Wordpress 或是其他線上平台，但是這些要嘛要維護主機，主機的費用價格也不便宜，而且像 Wordpress 那種使用資料庫 PHP 進行架站的，除了要顧好資料庫外，主機流量也需要消耗蠻多的，一點都不輕量。

平台式的如果哪天部落格平台忽然無預警倒閉等等，往往許多撰寫的內容都會付之一炬。

筆者後來找到一套適合來建置部落格的工具: Hugo

目前 Nasir 也是利用他來進行建置部落格。

使用起來真的超級輕量，不太會造成主機伺服器負擔，而且在撰寫的時候也是使用 Markdown 語法進行邊寫。

雖然可能在使用過程上步驟比較繁瑣，但是習慣後，加上後續的維護上，其實是個蠻方便的寫作平台！

說了這麼多，要如何開始＆進行安裝？

1. 安裝Hugo

首先我們要先在電腦上安裝 Hugo。

蘋果電腦打開終端機

Windows 電腦打開 CMD

brew install hugo

之後他就會在電腦中裝入 Hugo 工具了。

2. 建立一個新網站

安裝完後，要建立一個新的網站，在 CMD 或是終端機中輸入以下指令：

hugo new site my-blog

他就會出現在使用者的資料夾下。

3. 選擇並套用主題及模板

Hugo 有提供許多主題可以供使用者套用，可以到 https://themes.gohugo.io/ 去挑選喜歡的主題，然後複製主題的連結。


cd my-blog
git init
git submodule add https://github.com/your-theme-repo themes/your-theme

主題下載完成後，要到剛剛建立的部落格資料夾中，裡面找到一個 toml 的檔案，點開後把

theme = "your-theme" 改成自己主題的名字！

4. 開始建立部落格文章

hugo new posts/my-first-post.md

一樣也是在終端機或是 CMD 裡面進行輸入，輸入完後，會在 content/posts 裡建立一個 Markdown 的文件，就可以來編輯這個文件進行文章編輯。

5. 開啟本地端Server

由於是 Markdown 進行邊寫的，所以沒有所謂的即時預覽，所以需要開啟 Server 才能看到網頁呈現的樣子，這時可以利用 Hugo 提供的本地端 Server 進行預覽！

hugo server

這樣子就可以預覽網頁了。

6. 把部落格變成靜態網頁

hugo

輸入完後，就會在剛剛部落格資料夾裡面建立一個 public 的資料夾，這時生成的這個檔案就可以上傳到你的主機和伺服器上，像是 GitHub Page 等等靜態網頁主機！
